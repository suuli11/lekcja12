# jQuery DOM Traversal
Projekt pokazuje różne metody poruszania się po drzewie DOM w jQuery.

Funkcje aplikacji:
- parent() – zmiana koloru rodzica wybranego paragrafu
- parents() – zaznaczanie wszystkich przodków
- children() – podświetlenie dzieci div
- siblings() – podświetlenie rodzeństwa klikniętego elementu
- find() – kolorowanie span i ustawianie target="_blank" dla linków
- Nawigacja w drzewie DOM – przyciski pokazujące nazwy przodków i dzieci

Wykorzystane metody jQuery:
parent(), parents(), children(), siblings(), find(), click(), dblclick(), hover(), attr(), css(), map()
